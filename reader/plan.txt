Java Document Reader Application Plan
Overview
A simple Java Swing application that allows users to upload documents (TXT, EPUB, DOC, PDF) and read them in two modes: bionic reading (bold first 2-4 letters) and speed reading (configurable WPM up to 1000).

Technical Details
Bionic Reading Implementation
Split text into words
For each word, wrap first N letters (2-4) in <b> tags
Use JTextPane with HTML content type
Preserve line breaks and paragraphs
Speed Reading Implementation
Split text into words array
Calculate delay: 60000 / wpm milliseconds per word
Use javax.swing.Timer for word progression
Display one word at a time, centered
Large font (24-36pt) for readability
File Format Support
TXT: Direct file reading
PDF: PDFBox text extraction
EPUB: epub4j chapter extraction
DOC/DOCX: Apache POI text extraction
